Cloud Save tối giản

Game vẫn lưu trên thiết bị trước. Cloud chỉ được dùng khi bạn chủ động đăng nhập và bấm tải hoặc đẩy save.

1. Cấu hình Supabase

2. Đăng nhập email

Link đăng nhập sẽ quay lại đúng URL game. Email phải được cấu hình trong Supabase Auth.

3. Trạng thái

Chưa khởi tạo cloud.

Chưa đăng nhập.

4. Đồng bộ có kiểm soát

Trước khi ghi đè local bằng cloud, game tự tải một file backup. Khi phát hiện hai nhánh thay đổi, game dừng để tránh mất dữ liệu.

Ranh giới

Cloud Save chỉ lưu một JSON riêng tư theo tài khoản. Không có community, public profile, realtime hay multiplayer trong slice này.
